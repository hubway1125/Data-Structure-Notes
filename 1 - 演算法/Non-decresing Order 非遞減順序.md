# Non-decresing Order 非遞減順序
指非遞增順序 ( Ascending Order )，非遞增順序中不可有相同的數字出現，因為這樣違反了遞增的規則。而非遞增順序代表基本上他也是按照由小到大排列，但可能出現幾個相同的數字。

```
Strictly Ascending order: 1,2,3,4,5
Non-decreasing order: 1,2,2,3,3,4,5
```
